# 영화 마케팅 포인트 추출 Agent 실습과제

## 과제 목표

시놉시스와 관객 리뷰를 분석해 영화 홍보에 활용할 마케팅 포인트와 카피를 생성하는 Agent를 설계한다.

## 시놉시스 샘플

```text
지방 소도시의 오래된 극장에서 일하는 청년 민재는 폐관을 앞둔 극장을 지키기 위해 마지막 영화제를 준비한다. 
그러던 중 과거 이 극장에서 영화를 만들었던 은퇴 감독 수현을 만나고, 두 사람은 잊힌 필름을 복원하며 각자의 상처와 마주한다. 
영화제가 가까워질수록 마을 사람들은 극장에 얽힌 기억을 하나씩 꺼내고, 민재는 극장이 단순한 건물이 아니라 사람들의 시간을 품은 장소였음을 깨닫는다.
```

## 관객 리뷰 샘플

| 번호 | 리뷰 |
|---:|---|
| 1 | 잔잔하지만 오래 남는 영화였습니다. 극장이라는 공간이 주는 감성이 좋았어요. |
| 2 | 전개가 조금 느려서 중반부가 지루했습니다. |
| 3 | 배우들의 눈빛과 음악이 인상적이었습니다. |
| 4 | 큰 사건은 없지만 마음을 건드리는 장면이 많았습니다. |
| 5 | 예술영화에 익숙하지 않은 관객에게는 조금 심심할 수 있습니다. |

## 수행 단계

1. 시놉시스에서 장르, 핵심 갈등, 감정 톤을 추출한다.
2. 관객 리뷰에서 호평 요소와 불만 요소를 분류한다.
3. 타깃 관객을 2개 이상 정의한다.
4. 마케팅 포인트 3개를 도출한다.
5. 포스터 카피와 SNS 숏폼 문구를 생성한다.
6. 영화 톤과 맞지 않는 문구를 제거하고 개선한다.
