# 영화 마케팅 포인트 추출 Agent 커리큘럼

## 과정 개요

- 주제: 시놉시스와 관객 반응을 활용한 영화 마케팅 포인트 추출 Agent
- 대상: 영화 기획, 배급, 홍보, 콘텐츠 마케팅에 관심 있는 학습자
- 목표: 영화 정보를 구조화하고 타깃 관객, 핵심 카피, 홍보 포인트를 도출하는 Agent를 설계한다.
- 권장 시간: 3시간

## 학습 목표

- 시놉시스에서 장르, 갈등, 인물, 정서, 관객 소구점을 추출할 수 있다.
- 관객 리뷰에서 호평 요소와 불만 요소를 분류할 수 있다.
- 영화 홍보에 활용 가능한 카피와 숏폼 문구를 생성할 수 있다.
- 분석에서 마케팅 제안까지 이어지는 Workflow를 설계할 수 있다.

## 차시 구성

| 단계 | 주제 | 핵심 활동 | 산출물 |
|---:|---|---|---|
| 1 | 영화 정보 구조화 | 시놉시스 요소 추출 | 시놉시스 분석표 |
| 2 | 관객 반응 분석 | 리뷰 감정과 키워드 분류 | 관객 반응 요약 |
| 3 | 마케팅 포인트 도출 | 타깃, 소구점, 차별점 정리 | 마케팅 포인트 표 |
| 4 | 카피 생성 | 포스터 문구, SNS 문구 생성 | 카피 후보 |
| 5 | 발표 | 홍보 전략 공유 | Agent 설계 요약 |

## 권장 실습 시나리오

독립영화 배급사가 개봉 전 홍보 전략을 준비하고 있다. Agent는 시놉시스와 관객 리뷰를 입력받아 타깃 관객, 핵심 감정, 홍보 카피, 숏폼 콘텐츠 문구를 제안한다.

## 최종 산출물

- 시놉시스 분석표
- 관객 반응 분석표
- 마케팅 포인트 추출 프롬프트
- 홍보 카피 후보
- Agent 설계 요약서

## 평가 기준

| 항목 | 기준 |
|---|---|
| 분석 정확성 | 시놉시스의 핵심 갈등과 정서를 잘 추출했는가 |
| 관객 이해 | 타깃 관객과 소구점을 구체적으로 제안했는가 |
| 카피 품질 | 영화의 톤과 장르에 맞는 문구를 생성했는가 |
| Workflow 설계 | 분석과 제안 단계가 논리적으로 연결되는가 |
