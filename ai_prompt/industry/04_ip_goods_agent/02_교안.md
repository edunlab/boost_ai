# 콘텐츠 IP 굿즈 기획 Agent 교안

## 도입

질문으로 시작한다.

- "팬들이 사고 싶어 하는 굿즈는 단순히 로고가 붙은 상품일까요?"
- "IP의 어떤 요소가 상품으로 바뀔 때 매력적일까요?"

핵심 메시지:

> IP 굿즈 Agent는 상품을 많이 나열하는 도구가 아니라, 팬이 기억하고 소유하고 싶은 포인트를 상품으로 번역하는 도구다.

## 핵심 설명

### IP 분석 요소

- 세계관
- 주요 캐릭터
- 상징물
- 명대사 또는 장면
- 팬덤 밈
- 감정적 소유 욕구

### 굿즈 평가 기준

- 팬 매력도
- 제작 난이도
- 원가와 가격 적합성
- 일상 사용성
- 한정판 가능성
- IP 고유성

### 판매 전략 요소

- 타깃 고객
- 상품명
- 핵심 구매 이유
- 가격대
- 판매 채널
- 상세페이지 첫 문장
- 번들 구성

## 실습 진행

1. IP 샘플을 읽고 핵심 요소를 추출한다.
2. 굿즈 아이디어를 10개 이상 생성한다.
3. 평가 기준으로 상위 3개를 선정한다.
4. 최종 1개 상품의 상세 기획안을 작성한다.
5. 상세페이지 핵심 문구와 번들 전략을 만든다.
6. 팀별로 판매 가능성을 발표한다.

## 강사용 피드백 포인트

- 흔한 굿즈 나열보다 IP 고유성이 있는지 질문한다.
- 팬의 감정적 구매 이유를 반드시 적게 한다.
- 제작 난이도가 너무 높은 아이디어는 현실화 방안을 묻게 한다.
- 판매 문구는 세계관을 살리되 상품 정보도 명확해야 한다.

## 마무리 질문

- 이 굿즈는 왜 이 IP여야만 하는가?
- 팬이 이 상품을 소장하고 싶은 이유는 무엇인가?
- 커머스 상품으로 판매하려면 어떤 정보가 더 필요할까?
