# 콘텐츠 IP 굿즈 기획 Agent 실습과제

## 과제 목표

게임 또는 영화 IP를 분석해 굿즈 상품 아이디어, 상품성 평가, 판매 문구를 제안하는 Agent를 설계한다.

## IP 샘플

```text
IP명: 별빛 항로
장르: SF 어드벤처 게임
세계관: 인류가 잃어버린 별의 지도를 찾아 우주를 항해하는 이야기
주요 캐릭터: 초보 항해사 리안, 정비 로봇 모모, 전설의 함장 아르덴
상징 요소: 별의 지도, 낡은 우주선 키, 푸른 항성, 탐사 배지
팬덤 포인트: 모모의 귀여운 말투, 리안의 성장 서사, 별 지도를 완성하는 수집 경험
```

## 수행 단계

1. IP의 핵심 매력과 상징 요소를 추출한다.
2. 굿즈 아이디어 10개를 생성한다.
3. 팬 매력도, 제작 난이도, 가격 적합성, IP 고유성으로 평가한다.
4. 상위 3개 아이디어를 선정한다.
5. 최종 1개 상품의 상세 기획안을 작성한다.
6. 상세페이지 문구와 판매 전략을 만든다.
