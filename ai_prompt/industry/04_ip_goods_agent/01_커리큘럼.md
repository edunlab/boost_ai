# 콘텐츠 IP 굿즈 기획 Agent 커리큘럼

## 과정 개요

- 주제: 게임/영화 IP를 활용한 굿즈 상품 기획 Agent 설계
- 대상: 콘텐츠 기획, IP 비즈니스, 머천다이징, 커머스에 관심 있는 학습자
- 목표: IP의 세계관과 팬 특성을 분석해 굿즈 아이디어와 판매 전략을 제안하는 Agent를 설계한다.
- 권장 시간: 3시간

## 학습 목표

- IP의 핵심 매력, 캐릭터, 세계관, 팬덤 요소를 구조화할 수 있다.
- IP 특성에 맞는 굿즈 상품 아이디어를 제안할 수 있다.
- 타깃 고객, 가격대, 판매 채널, 상세페이지 메시지를 설계할 수 있다.
- 콘텐츠와 커머스를 연결하는 Agent Workflow를 만들 수 있다.

## 차시 구성

| 단계 | 주제 | 핵심 활동 | 산출물 |
|---:|---|---|---|
| 1 | IP 분석 | 세계관, 캐릭터, 팬덤 포인트 추출 | IP 분석표 |
| 2 | 굿즈 아이디어 생성 | 상품군과 콘셉트 제안 | 굿즈 후보 리스트 |
| 3 | 상품성 평가 | 제작 난이도, 팬 매력도, 가격대 평가 | 우선순위표 |
| 4 | 판매 전략 설계 | 상세페이지, 번들, 채널 전략 | 판매 전략안 |
| 5 | 발표 | IP 굿즈 기획안 공유 | Agent 설계 요약 |

## 권장 실습 시나리오

게임 또는 영화 IP를 활용해 팬 대상 굿즈를 출시하려 한다. Agent는 IP 정보를 분석하고 굿즈 후보를 생성한 뒤, 상품성 평가와 판매 문구까지 제안한다.

## 최종 산출물

- IP 분석표
- 굿즈 아이디어 후보
- 상품성 평가표
- 상세페이지 핵심 문구
- Agent 설계 요약서

## 평가 기준

| 항목 | 기준 |
|---|---|
| IP 이해 | 세계관과 팬덤 포인트를 잘 반영했는가 |
| 상품성 | 실제 판매 가능한 굿즈 아이디어인가 |
| 차별성 | IP 고유성이 상품에 드러나는가 |
| 판매 전략 | 타깃, 가격, 채널, 메시지가 구체적인가 |
