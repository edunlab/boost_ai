# 커머스 상품 리뷰 개선 Agent 실습과제

## 과제 목표

상품 리뷰를 분석해 상세페이지 개선 문구, FAQ, CS 응답 초안을 만드는 Agent를 설계한다.

## 상품 정보

```text
상품명: 접이식 무선 미니 선풍기
특징: 휴대용, 3단 풍속 조절, USB-C 충전, 최대 8시간 사용, 책상 거치 가능
가격대: 29,900원
주요 고객: 학생, 직장인, 야외 활동이 많은 고객
```

## 리뷰 샘플

| 번호 | 리뷰 |
|---:|---|
| 1 | 작고 가벼워서 출퇴근할 때 들고 다니기 좋습니다. |
| 2 | 생각보다 바람이 약해서 야외에서는 아쉽습니다. |
| 3 | 충전 단자가 USB-C라 편하지만 배터리가 8시간까지는 안 가는 것 같아요. |
| 4 | 배송 중 박스가 찌그러져서 왔습니다. 제품은 괜찮았어요. |
| 5 | 책상에 세워두기 편해서 사무실에서 잘 쓰고 있습니다. |
| 6 | 소음이 조금 있어서 조용한 도서관에서는 쓰기 어렵습니다. |

## 수행 단계

1. 리뷰를 품질, 배송, 가격, 사용성, CS, 재구매 의사로 분류한다.
2. 반복되는 불만 원인을 정리한다.
3. 상세페이지에 추가할 안내 문구를 작성한다.
4. FAQ 3개를 만든다.
5. CS 응답 초안 2개를 만든다.
6. 개선 우선순위를 정한다.
