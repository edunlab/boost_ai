# 커머스 상품 리뷰 개선 Agent 커리큘럼

## 과정 개요

- 주제: 상품 리뷰를 분석해 상세페이지, FAQ, CS 대응을 개선하는 Agent 설계
- 대상: 커머스 MD, 마케터, CS 담당, 상품 기획에 관심 있는 학습자
- 목표: 상품 리뷰를 분류하고 구매 전환과 고객 만족 개선에 필요한 액션을 도출한다.
- 권장 시간: 3시간

## 학습 목표

- 상품 리뷰를 품질, 배송, 가격, 사용성, CS, 재구매 의사로 분류할 수 있다.
- 리뷰 속 불만 원인을 상세페이지 개선 포인트로 변환할 수 있다.
- 고객 질문과 불만을 FAQ 및 CS 응답 초안으로 만들 수 있다.
- 커머스 운영 업무에 적합한 Agent 흐름을 설계할 수 있다.

## 차시 구성

| 단계 | 주제 | 핵심 활동 | 산출물 |
|---:|---|---|---|
| 1 | 리뷰 데이터 이해 | 구매자 리뷰 유형 분류 | 리뷰 분류 기준표 |
| 2 | 불만 원인 분석 | 반복 이슈와 구매 장벽 도출 | 개선 포인트 표 |
| 3 | 상세페이지 개선 | 상품 설명, FAQ, 고지 문구 제안 | 상세페이지 문구 |
| 4 | CS 응답 생성 | 상황별 응답 초안 작성 | CS 응답 템플릿 |
| 5 | 발표 | 전환 개선 관점 공유 | Agent 설계 요약 |

## 권장 실습 시나리오

온라인 쇼핑몰의 신상품 리뷰가 쌓이고 있다. Agent는 리뷰를 분석해 고객이 불편해하는 지점을 찾고, 상세페이지 문구와 FAQ, CS 답변을 개선하도록 제안한다.

## 최종 산출물

- 리뷰 분류 기준표
- 상품 리뷰 분석 결과표
- 상세페이지 개선 문구
- FAQ/CS 응답 초안
- Agent 설계 요약서

## 평가 기준

| 항목 | 기준 |
|---|---|
| 리뷰 분류 | 커머스 업무에 맞는 분류 기준을 세웠는가 |
| 개선 연결 | 리뷰 분석이 상세페이지나 CS 개선으로 이어지는가 |
| 출력 활용성 | MD, 마케터, CS 담당자가 바로 쓸 수 있는가 |
| 예외 처리 | 악성 리뷰, 복합 리뷰, 정보 부족 리뷰를 다루는가 |
