# 게임 유저 리뷰 분석 Agent 실습과제

## 과제 목표

게임 유저 리뷰를 자동으로 분류하고, 운영팀의 후속 조치를 제안하는 Agent를 설계한다.

## 리뷰 샘플

| 번호 | 리뷰 |
|---:|---|
| 1 | 업데이트 이후 접속하면 바로 튕깁니다. 보상도 못 받았어요. |
| 2 | 신규 캐릭터 디자인이 너무 좋고 스토리도 몰입감 있습니다. |
| 3 | 뽑기 확률이 너무 낮아서 과금할 의욕이 사라졌습니다. |
| 4 | 보스 난이도가 갑자기 올라서 신규 유저는 진행이 어렵습니다. |
| 5 | 길드전 말고 친구랑 같이 할 수 있는 협동 콘텐츠가 더 있으면 좋겠습니다. |
| 6 | 게임은 재미있는데 로딩이 길고 발열이 심합니다. |

## 수행 단계

1. 리뷰 분류 카테고리를 정의한다.
2. 각 카테고리의 판단 기준을 작성한다.
3. Agent 프롬프트를 작성한다.
4. 리뷰 샘플을 입력해 분류 결과를 생성한다.
5. 긴급도와 후속 조치를 추가한다.
6. 잘못 분류된 사례를 찾아 프롬프트를 개선한다.
