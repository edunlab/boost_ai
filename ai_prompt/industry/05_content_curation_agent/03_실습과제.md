# 콘텐츠 추천 큐레이션 Agent 실습과제

## 과제 목표

사용자 취향과 상황을 입력받아 게임, 영화, 상품을 묶어 추천하는 큐레이션 Agent를 설계한다.

## 사용자 프로필 샘플

```text
사용자 A
- 주말 저녁에 3시간 정도 여유가 있음
- 몰입감 있는 스토리를 좋아함
- 너무 잔인하거나 공포가 강한 콘텐츠는 싫어함
- 혼자 즐길 예정
- 예산은 3만 원 이하
- 최근 좋았던 콘텐츠: 감성적인 SF 영화, 서사 중심 어드벤처 게임
```

## 수행 단계

1. 사용자 프로필에서 추천 조건을 추출한다.
2. 게임, 영화, 상품 추천 기준을 각각 작성한다.
3. 추천 결과를 1순위와 대체안으로 제시한다.
4. 추천 이유를 사용자 조건과 연결해 설명한다.
5. 추천하지 않는 유형과 제외 이유를 작성한다.
6. 다른 사용자 프로필을 만들어 결과 차이를 테스트한다.
