# 콘텐츠 추천 큐레이션 Agent 커리큘럼

## 과정 개요

- 주제: 사용자 취향 기반 게임·영화·상품 추천 Agent 설계
- 대상: 콘텐츠 플랫폼, 커머스 추천, 서비스 기획에 관심 있는 학습자
- 목표: 사용자의 취향 정보를 입력받아 게임, 영화, 상품을 묶어 추천하고 추천 이유를 설명하는 Agent를 설계한다.
- 권장 시간: 3시간

## 학습 목표

- 사용자 취향 정보를 추천 조건으로 구조화할 수 있다.
- 게임, 영화, 상품 추천 기준을 구분해 설계할 수 있다.
- 추천 결과에 이유, 주의점, 대체안을 함께 제시할 수 있다.
- 추천 Agent의 조건 분기와 출력 형식을 설계할 수 있다.

## 차시 구성

| 단계 | 주제 | 핵심 활동 | 산출물 |
|---:|---|---|---|
| 1 | 취향 데이터 구조화 | 사용자 프로필과 선호 조건 정리 | 취향 입력표 |
| 2 | 추천 기준 설계 | 장르, 분위기, 가격, 이용 상황 정의 | 추천 기준표 |
| 3 | 큐레이션 생성 | 게임·영화·상품 묶음 추천 | 추천 결과표 |
| 4 | 설명 가능성 강화 | 추천 이유와 제외 이유 작성 | 설명 프롬프트 |
| 5 | 발표 | 큐레이션 콘셉트 공유 | Agent 설계 요약 |

## 권장 실습 시나리오

사용자가 자신의 취향과 상황을 입력하면 Agent가 주말에 즐길 게임, 영화, 관련 상품을 하나의 큐레이션 세트로 추천한다. 추천 결과에는 이유와 대체안이 포함된다.

## 최종 산출물

- 사용자 취향 입력표
- 추천 기준표
- 큐레이션 결과표
- 추천 설명 프롬프트
- Agent 설계 요약서

## 평가 기준

| 항목 | 기준 |
|---|---|
| 취향 구조화 | 사용자 입력을 추천 조건으로 잘 변환했는가 |
| 추천 일관성 | 게임, 영화, 상품이 하나의 콘셉트로 연결되는가 |
| 설명 가능성 | 추천 이유가 구체적이고 납득 가능한가 |
| 조건 분기 | 예산, 시간, 장르 선호에 따라 결과가 달라지는가 |
